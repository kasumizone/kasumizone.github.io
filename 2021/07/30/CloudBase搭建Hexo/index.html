静态网站托管服务选择

说起静态网站托管服务，我想有了解过Hexo和静态博客，有建站打算的人儿们都有了解过：Github Page，Gitee Page…等

我们下面针对这两种基于Git的远程仓库提供的静态网站托管服务进行优劣的简单分析和服务选择~

Gitee Page

作为国内的Github，稳定性和速度都毋庸置疑

作为世界上最大的“同性交友网站”，对国内来说相当不稳定，有些地区甚至完全无法访问，各种网络限制也很严重，需要大家都能够科学上网，才能勉强使用~

Hexo简介

Hexo 是一个基于nodejs的静态博客网站生成器，是目前非常流行的静态博客，想必想搭博客的小伙伴应该早有耳闻~

Hexo的优势很明显，即可以很好的部署在静态网站托管服务上，无需购买服务器；再者，静态博客框架中，Hexo有着悠久的历史和庞大的用户量，所以质量肯定没话说，主题也是相当之多，官方的主题库，大神自己写的主题，都相当华丽，定能够满足大多数人的喜好~

Hexo使用文档

在这里再特别跟大家推荐一个博客系统：Halo

（可能）被誉为当下最优秀的博客系统~

之前有使用过Halo，总体来说非常棒，体验相当好，使用Java开发，并且代码开源，易于部署↓

最重要的是，有很方便的后台管理系统（吹爆），各种管理操作都非常方便，不用很繁琐的处理标签，友链，文章等数据，对非计算机专业的孩子们非常友好，存在的问题就是需要购买服务器部署，目前主题的数量也不是很多，但如果大家有服务器也很推荐去使用Halo，非常优秀，未来可期！

环境准备

Git（选择性使用）

为什么是选择性使用？如果我们选择的是Gitee Page或Github配置，那么Git是必要环境，但我们这里使用的是CloudBase，所以不进行动态部署的话，是没有必要使用Git的~

Git的安装和教程网上有很多，我就不多赘述了，不是懒，是不配 …..((/- -)/

NodeJS

官方网站下载并安装 …..((/- -)/

安装完成后打开命令行输入以下命令查看是否安装成功~

node -v
npm -v

输入以下命令进行淘宝镜像的配置

npm install -g cnpm --registry=https://registry.npm.taobao.org

接着，下面的命令凡是npm均可替换为cnpm，使用淘宝镜像去下载资源，快了不止一点~

安装与初始化Hexo

安装

npm install -g hexo-cli

使用hexo -v可查看版本信息

初始化

hexo init [文件名]
# 不写文件名则默认再当前目录进行初始化，且当前目录必须是空目录
# 加了文件名则会再当前目录创建与文件名相同的文件夹并在生成的文件夹下进行初始化
npm install

启动Hexo

下面就到第一个验证结果的环节

先进入到Hexo初始化的文件夹下，执行下面的命令

hexo g
hexo server

即可启动Hexo服务，默认在本地4000端口启动，现在访问localhost:4000即可看到初步的成效~

想自定义域名的看这里~

CloudBase帮我们自动生成的域名实在太长，我们自己看着还好，出去分享属实有点拿不出手o(一︿一+)o

那么就需要我们自定义域名，国内比较出名的域名服务商，比如阿里云，腾讯云，华为云都可以去购买域名，如果大家是第一次购买，应该都会有1元1年购买域名的福利，只是后缀比较可惜，不过问题应该不大 (๑ˉ∀ˉ๑)

CloudBase是腾讯云提供的服务，相关配置也是根据腾讯云域名进行讲解的，我这里用的是阿里云域名，具体的域名绑定步骤如下：

备案

首先域名是要备案的，没有备案过的小伙伴可以先去备案，这个过程相对复杂且漫长，并且为了安全起见，备案的前提是要有你买的域名的服务商的服务器（感觉很坑）

具体的步骤可以去你买的域名的官方平台查看。个人感觉阿里云的域名备案要方便的多，听说腾讯云要使用备案专用的背景布拍照上传(⊙ˍ⊙)

SSL证书

自定义域名时，需要提供域名的SSL证书，使域名可以用HTTPS进行访问，SSL证书的购买是很贵的，（很惭愧这方面我也不是特别了解），我的方法是这样：

首先去到SSL证书的首页

点击选购SSL证书

点击如图所示的免费试用，然后一路点击购买即可，均是免费的，并且秒签发，非常给力~

随后进入SSL证书控制台，按如下步骤进行签发申请和下载操作，一路行云流水~

随后入腾讯云的SSL证书控制台，将刚刚下载下来的证书上传，按照下图所示进行填写即可~

接着按照官方文档的指示，添加自定义域名，域名输入后，会自动识别填充证书~

接着根据提示，拿到.cdn.dnsv1.com结尾的域名，前往阿里云域名控制台配置CNAME域名解析

最后等待部署完成，即可通过自己的域名访问Hexo了，具体Hexo的使用及主题，参考官方文档即可~
